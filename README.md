# 📚 Library Quest - สื่อการเรียนรู้ประเภทห้องสมุด (ม.1)

สื่อการเรียนรู้รายวิชาการใช้ห้องสมุดเบื้องต้น เรื่อง **"ประเภทของห้องสมุด"** สำหรับนักเรียนชั้นมัธยมศึกษาปีที่ 1 ตกแต่งด้วยดีไซน์สีม่วง-ขาว สดใส สนุกสนาน และทันสมัย พร้อมลูกเล่นมินิเกมจำลองสถานการณ์ แบบทดสอบวัดความรู้ตาม Bloom's Taxonomy และระบบออกเกียรติบัตรออนไลน์แบบดาวน์โหลดได้จริง เพื่อนำไปจัดเผยแพร่ (Deploy) บน GitHub Pages ของคุณ:
👉 [https://dreamiieboon-tech.github.io/](https://dreamiieboon-tech.github.io/)

---

## 🌟 คุณสมบัติเด่นของเว็บไซต์ (Key Features)

1. **บทเรียนประเภทห้องสมุดทั้ง 5 (interactive Learning Tabs)**:
   - นำเสนอข้อมูลห้องสมุด 5 ประเภท ได้แก่ ห้องสมุดโรงเรียน, ห้องสมุดมหาวิทยาลัย, ห้องสมุดเฉพาะ, ห้องสมุดประชาชน, และหอสมุดแห่งชาติ
   - ให้ข้อมูลครบถ้วนละเอียดยิบตามหลักสูตร ม.1: ความหมาย, ลักษณะเด่น, วิธีการใช้งานอย่างถูกต้อง, บริการที่มอบให้, และตัวอย่างที่เข้าใจง่าย
   - นำเสนอในรูปแบบแท็บอินเตอร์แอคทีฟ คลิกเปลี่ยนเนื้อหาและภาพประกอบอย่างรวดเร็วและลื่นไหล
2. **มินิเกมผู้พิทักษ์ห้องสมุด (Library Guardian Game)**:
   - เกมจับคู่สั้น ๆ 5 ด่านที่แสดงโจทย์สถานการณ์ชีวิตประจำวันของผู้มาใช้งานห้องสมุดประเภทต่าง ๆ
   - ฝึกให้นักเรียนวิเคราะห์และเลือกว่าควรแนะนำให้พวกเขาไปห้องสมุดประเภทใด
   - แสดงผลแบบ Real-time พร้อมอธิบายเหตุผลประกอบ
3. **แบบทดสอบตาม Bloom's Taxonomy (10 Questions Quiz)**:
   - ข้อสอบ 10 ข้อ เรียงลำดับจากระดับพฤติกรรมการเรียนรู้ง่ายไปยาก (ความจำ -> ความเข้าใจ -> ประยุกต์ใช้ -> วิเคราะห์ -> ประเมินค่า -> สร้างสรรค์)
   - ระบบตรวจคะแนนอัตโนมัติพร้อมเฉลยและอธิบายความรู้อย่างละเอียดทันทีหลังจากตอบแต่ละข้อ
   - ออกแบบระดับภาษาที่เหมาะกับนักเรียนวัยรุ่นตอนต้น ม.1 อ่านง่ายและสนุกสนาน
4. **เกียรติบัตรออนไลน์อัจฉริยะ (E-Certificate Canvas Generator)**:
   - เมื่อทำคะแนนผ่านเกณฑ์ 6 ข้อขึ้นไป (60%) ระบบจะปลดล็อคเกียรติบัตร
   - นำชื่อผู้สอบและคะแนนไปวาดลงบนแผ่นเกียรติบัตรด้วยเทคนิค HTML5 Canvas
   - มีปุ่มดาวน์โหลดเป็นรูปภาพ PNG ทันที หรือกดสั่งพิมพ์ (Print/Save to PDF) ได้ทันที

## 🎨 การออกแบบและโทนสี (Design Aesthetics)

- **สี (Colors)**: โทนสีม่วง-ขาว สดใส (Vibrant Purple & Clean White) นำสายตาด้วยสีม่วงเข้มและม่วงอ่อนสลับกัน เพิ่มความกระปรี้กระเปร่าด้วยสีทองส้มของดวงดาวและเกียรติบัตร
- **สไตล์ (Layout)**: เรียบง่าย สะอาดตา การ์ดคอนเทนต์แบบมนกลมเงาซอฟต์ และแอนิเมชันลูกเล่นการลอยตัวและการเปลี่ยนผ่าน (Smooth Transitions)
- **ตัวอักษร (Fonts)**: Google Fonts `Kanit` (สำหรับหัวข้อ) และ `Sarabun` (สำหรับเนื้อความตัวเล็ก) ทำให้อ่านง่าย สบายตา สอดรับกับผู้ใช้ที่เป็นเยาวชนและครูผู้สอน

## 💻 วิธีการเปิดใช้งานเครื่องตนเองเพื่อทดสอบ (Local Setup)

1. ดาวน์โหลดโฟลเดอร์นี้ไว้ในคอมพิวเตอร์ของคุณ
2. ดับเบิลคลิกเปิดไฟล์ `index.html` ผ่านบราวเซอร์ยอดนิยม เช่น Google Chrome, Microsoft Edge, หรือ Safari ได้ทันทีโดยไม่ต้องรันเซิร์ฟเวอร์ยุ่งยาก
3. หรือรันเซิร์ฟเวอร์พัฒนาขนาดเล็กด้วย VS Code Live Server หรือสั่งงานผ่าน Node:
   ```bash
   npx serve .
   ```

## 🚀 วิธีการอัปโหลดขึ้น GitHub Pages (Deployment Guide)

เพื่ออัปโหลดโค้ดชุดนี้ไปแสดงผลบนหน้าเว็บไซต์ของคุณที่ `https://dreamiieboon-tech.github.io/` ให้ปฏิบัติตามคำสั่งของ Git ในโฟลเดอร์โครงการนี้ดังนี้:

1. เปิด Terminal หรือ PowerShell ในตำแหน่งโฟลเดอร์โครงการนี้ (`c:\Users\HP\Desktop\dream`)
2. เริ่มต้นระบบกิก (Git) และตั้งค่าปลายทางของ Repository:
   ```bash
   git init
   git remote add origin https://github.com/dreamiieboon-tech/dreamiieboon-tech.github.io.git
   ```
3. เพิ่มไฟล์เข้าระบบและทำ Commit ครั้งแรก:
   ```bash
   git add .
   git commit -m "Initial commit - Library Quest interactive media for Grade 7"
   ```
4. เปลี่ยนชื่อ Branch หลักให้เป็น `main` และทำการผลักข้อมูล (Push) ขึ้นคลาวด์:
   ```bash
   git branch -M main
   git push -u origin main
   ```
5. หลังจากผลักข้อมูลสำเร็จ รอระบบ GitHub ดำเนินการสร้างเว็บไซต์ (Build) ประมาณ 1-2 นาที จากนั้นเปิดดูผ่านลิงก์ `https://dreamiieboon-tech.github.io/` ได้ทันที!
